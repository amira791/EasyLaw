المادة الأولى

لأغراض هذا الاتفاق يقصد ب :

1 - "إدارات الجمارك" :

بالنّسبة لحكومة الجمهوريّة الجزائريّة
الديمقراطيّة الشٌعبيَّة : المديرية العامة للجمارك.

بالنّسبة لحكومة جمهوريّة تركيا : الوزارة الأولى
نيابة كتابة الدولة للجمارك.

2- "التشريع الجمركي" : مجموع الأحكام
التشريعية والتنظيمية المتعلّقة بالاستيراد والتصدير
والمسافنة والعبور والتخزين وتداول البضائع والتي
تتكفقّل بتطبيقها إدارات الجمارك للأطراف المتعاقدة
وكذا الأنظمة المت علّقة بالحظر والقيود والرقابة
المماثلة عند الحدود.

3- "المخالفة الجمركية” : كلّ خرق أو محاولة
خرق للتشريع الجمركي.

4- "شخص” : كل شخص طبيعي أو معنوي.

5-" لمعطيات ذات الطابع الشخصي" :
المعطيات الخاصة بشخص طبيعي هويته معروفة
أو يمكن التعرف عليها.

6 - "المعلومات" : كلّ معطيات أو وثيقة أو تقرير
أو نسخ منها مصادق عليها أو أي تبليغ آخر.

7- "الاستعلامات" : كل المعلومات المعالجة
أو التي يتم تحليلها قصد الإدلاء بتوضيحات خاصّة
بمخالفة جمركية.

8 - "الإدارة الطالبة” : إدارة الجمارك التي تقدّم
طلب المساعدة.

9- "الإدارة المطلوبة” : إدارة الجبمارك التي
تتسلّم طلب المساعدة المرسل إليها.
