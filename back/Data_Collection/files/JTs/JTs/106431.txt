المادّة 3 : يقصد. في مفهوم هذا الأمر ؛ بما يأتي :

1- الموظف العمومي :

- كل شخص يشغل منصبا تشريعيا أو تنفيذيا أو إداريا

أو غير مدفوع الأجر ء بصرف النظر عن رتبته أو أقدميته؛

- كل شخص آخر يتولى. ولو مؤقتاء وظيفة أو وكالة
بأجر أو بدون أجرء ويساهم بهذه الصفة في خدمة هيئة
عمومية أو مؤسسة عمومية أو أي مؤسسة أخرى تملك

- كل شخص آخر معرّف بأنه موظف عمومي أو من في
حكمه طبقا للتشريع والتنظيم المعمول بهماء

2- الوثيقة : المراسلات والمحررات والمستندات التى
أنشأتها أو حصلت عليها أي من السلطات المعنية أثناء
ممارسة نشاطهاء

3- الوثائق المصنفة : أي مكتوب ورقي أو إلكتروني
أو رسم أو مخطط أو خريطة أو صورة أو شريط صوتي أو
سمعي بصري أو أي سند مادي أو إلكتروني آخر كانت
محل تدابير ترمي إلى منع نشرها أو تقييد الاطلاع عليهاء

4- المعلومات : أي حدث أو خبر مهما كان مصدره.؛ وثيقة
أو صورة أو شريط صوتي أو مرئّي أو سمعي بصري أو
محادشة أو مكالمة هاتفية, يؤدى الكشف عنها إلى المساس
بالسلطات المعنية. ‎١‏
