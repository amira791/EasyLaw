ورقلة والمحاكم التّابعة له كما يأتي :

- محكمة ورقلة : سبعة مكاتب

- محكمة توقرت : أربعة مكاتب

- محكمة إيليزي : مكتبان

- محكمة جانت : مكتبان.

المادة 32 : يحدد مقر وعدد المكاتب العموميّة
للتّوثيق التي تشملها دائرة اختصاص مجلس قضاء
وهران والمحاكم التّابعة له كما يأتي :

- محكمة وهران : اثنان وثلاثون مكتبا

- محكمة أرزيو : خمسة مكاتب

- محكمة المرسى الكبير : ثلاثة مكاتب

- محكمة السنّائية : ستّة مكاتب ٍ

- محكمة وادي تليلات : ثلاثة مكاتب

- محكمة قديل : ثلاثة مكاتب.
